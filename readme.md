# Cloudflared Tunnel Configuration Repository

This repository documents configurations and best practices for exposing a local web site using Cloudflare Tunnel (cloudflared).

Index
-----
- Cloudflared setup and examples: [cloudflared/configuration.md](cloudflared/configuration.md)
- Quick setup summary: `cloudflared-setup.txt`
- Firewall rules documentation: [firewalls/rules.md](firewalls/rules.md)
- Firewall summary: `firewalls/rules.txt`
- Firewall rules created by my: [firewalls/extra_rules.md](firewalls/extra_rules.md)

Repository structure
--------------------
- `cloudflared/` — Cloudflared configuration examples and instructions
  - `configuracion.md` — Detailed setup and screenshots placeholder
  - `cloudflared.yaml` — Example Cloudflared configuration file
- `firewalls/` — Firewall rules, YAML examples and summaries
  - `rules.md` — Explanation and screenshots placeholder
  - `rules.yaml` — YAML rule examples
  - `rules.txt` — Short summary linking to `rules.md`
  - `extra_rules.md` — More updated rules `extra_rules.md`
